## 服务端依赖 (Ubuntu 22.04)

### 系统级依赖
```bash
# 基础工具
sudo apt-get update
sudo apt-get install -y python3 python3-pip git curl

# Manim 依赖
sudo apt-get install -y ffmpeg libcairo2-dev libpango1.0-dev \
    pkg-config python3-dev texlive texlive-latex-extra \
    texlive-fonts-recommended texlive-fonts-extra \
    texlive-lang-chinese cm-super dvipng

# GPU 支持（可选）
sudo apt-get install -y nvidia-cuda-toolkit
```

### Python 包（服务端）
```bash
pip3 install fastapi uvicorn[standard] pydantic python-multipart
pip3 install manim>=0.19.0
pip3 install dashscope librosa  # TTS 相关
pip3 install requests
```

### 服务配置
```bash
# 创建 systemd 服务
sudo systemctl enable manim.service
sudo systemctl start manim.service
```

## 快速安装（服务端）

```bash
#!/bin/bash
# 一键安装脚本

# 1. 系统依赖
sudo apt-get update
sudo apt-get install -y python3 python3-pip git curl ffmpeg \
    libcairo2-dev libpango1.0-dev pkg-config python3-dev \
    texlive texlive-latex-extra texlive-fonts-recommended \
    texlive-fonts-extra texlive-lang-chinese cm-super dvipng

# 2. Python 依赖
pip3 install fastapi uvicorn[standard] pydantic python-multipart
pip3 install manim>=0.19.0
pip3 install dashscope librosa requests

# 3. 创建服务目录
sudo mkdir -p /home/ubuntu/math/manim-service
cd /home/ubuntu/math/manim-service

# 4. 配置 systemd 服务（参考 main.py 中的配置示例）
```

## 快速安装（客户端）

```bash
# 创建虚拟环境
python3 -m venv .venv
source .venv/bin/activate

# 安装依赖
pip install requests python-dotenv

# 配置环境变量
echo "MANIM_SERVICE_URL=http://你的服务器IP:8000" > .env
```

## 注意事项

1. **中文字体**：确保服务器安装了中文字体支持
   ```bash
   sudo apt-get install fonts-wqy-zenhei fonts-wqy-microhei
   ```

2. **GPU 加速**：如需 GPU 加速，安装 NVIDIA 驱动和 CUDA
   ```bash
   sudo apt-get install nvidia-driver-525 nvidia-cuda-toolkit
   ```

3. **DashScope API**：需要阿里云 DashScope API Key
   - 注册地址：https://dashscope.aliyun.com/
   - 配置：`export DASHSCOPE_API_KEY="your-api-key"`

4. **防火墙**：确保服务器 8000 端口开放
   ```bash
   sudo ufw allow 8000
   ```
