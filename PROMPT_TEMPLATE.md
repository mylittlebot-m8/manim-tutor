# Tutor Skill - 分镜脚本生成提示词模板

## 当前使用的提示词

### 基础版本（4 幕）

```markdown
你是一位专业的数学教学视频设计师。请根据以下数学题目，生成详细的分镜脚本。

## 题目
{problem_text}

## 要求

生成一个完整的分镜脚本，包含以下内容：

### 第 1 幕：题目朗读
- 画面：深色背景，题目文字逐行显示
- 字幕：题目标题
- 读白：完整朗读题目（约 20-30 秒）
- 动画：标题淡入，题目条件逐行显示
- 目的：引入题目，让学生了解问题背景

### 第 2 幕：问题分析
- 画面：列出已知条件和求解目标
- 字幕："问题分析"
- 读白：分析已知条件和求解目标（约 10-15 秒）
- 动画：已知条件列表显示，求解目标高亮
- 目的：理清思路

### 第 3 幕：建立模型
- 画面：建立数学模型或方程
- 字幕："建立模型"
- 读白：说明建模思路（约 10-15 秒）
- 动画：变量设定，方程或关系式显示
- 目的：数学建模

### 第 4 幕：求解过程
- 画面：详细求解步骤
- 字幕："求解过程"
- 读白：逐步讲解求解过程（约 20-30 秒）
- 动画：第一步计算，第二步计算，得出结果
- 目的：展示完整求解过程

## 输出格式

使用 Markdown 格式，每个分镜包含：
- 画面描述
- 字幕内容
- 读白文本（用于 TTS 生成）
- 动画时间轴
- 设计目的

## 音频清单

最后生成音频文件清单表格：
| 幕号 | 文件名 | 读白文本 | 时长 | 说话人 | 情感 |
|------|--------|----------|------|--------|------|
| 1 | audio_001_scene.wav | ... | | Nofish | 平和 |
| 2 | audio_002_scene.wav | ... | | Nofish | 平和 |
| ... | ... | ... | | ... | ... |
```

## 完整版本（8 幕）- 推荐

```markdown
你是一位专业的数学教学视频设计师。请根据以下数学题目，生成详细的分镜脚本。

## 题目
{problem_text}

## 题型
{problem_type}  # 如：抛物线综合题、函数应用题、几何证明题

## 难度
{difficulty}  # 1-5 星

## 要求

生成一个完整的分镜脚本，包含 8 幕内容：

### 第 1 幕：题目展示（约 20-30 秒）
- 画面：深色背景，题目文字逐行显示
- 字幕：题目标题
- 读白：完整朗读题目，包括所有小问
- 动画：标题淡入，抛物线方程显示，三个问题逐行显示
- 目的：引入题目，让学生了解问题背景

### 第 2 幕：条件分析（约 15-20 秒）
- 画面：列出已知条件和求解目标
- 字幕："条件分析"
- 读白：分析已知条件（抛物线方程含参数 m，二次函数，开口向上），说明各问考查内容
- 动画：已知条件列表显示，抛物线开口方向标注，三个求解目标高亮
- 目的：理清思路，明确解题方向

### 第 3 幕：第 1 问求解（约 15-20 秒）
- 画面：展示顶点公式和代入过程
- 字幕："第 (1) 问：求顶点坐标"
- 读白：详细讲解顶点公式代入和化简过程
- 动画：显示顶点公式，代入 a,b 值，化简得 x=m+2，代入求 y，显示最终结果
- 目的：展示顶点坐标的完整推导过程

### 第 4 幕：第 2 问求解（约 20-25 秒）
- 画面：展示代入 A、B 点和解不等式
- 字幕："第 (2) 问：求 m 范围"
- 读白：讲解代入法和不等式求解过程
- 动画：代入 A 点，化简得 a 的表达式，代入 B 点，化简得 b 的表达式，列出不等式，解出 m 的范围
- 目的：展示代入法和不等式求解过程

### 第 5 幕：第 3 问求解（约 25-30 秒）
- 画面：展示对称轴位置和分类讨论
- 字幕："第 (3) 问：最大值 - 最小值 = 4"
- 读白：讲解分类讨论思想，根据对称轴与区间位置关系分三种情况
- 动画：显示对称轴公式，显示给定区间，说明开口方向，情况 1(m<-3)，情况 2(m>1)，情况 3(-3≤m≤1)，说明求解方法
- 目的：展示分类讨论思想和解题思路

### 第 6 幕：验证答案（约 10-15 秒）【新增】
- 画面：代入验证各问答案
- 字幕："验证答案"
- 读白：逐问验证答案正确性
- 动画：第 1 问验证，第 2 问验证（取特值），第 3 问验证，显示"答案正确"
- 目的：培养学生验证答案的习惯

### 第 7 幕：方法总结（约 15-20 秒）【新增】
- 画面：回顾解题方法和关键点
- 字幕："方法总结"
- 读白：总结各问考查的知识点和解题方法
- 动画：第 1 问方法回顾（顶点公式），第 2 问方法回顾（代入法 + 不等式），第 3 问方法回顾（分类讨论），显示关键点
- 目的：巩固知识点，提升解题能力

### 第 8 幕：拓展思考（约 15-20 秒）【新增】
- 画面：提出变式问题和注意事项
- 字幕："拓展思考"
- 读白：提出变式问题，强调解题关键
- 动画：变式问题 1（开口向下），变式问题 2（区间改变），解题关键（画图）
- 目的：拓展思维，举一反三

## 输出格式

使用 Markdown 格式，每个分镜包含：
- **画面**：详细的视觉描述
- **字幕**：屏幕显示的文字
- **读白**：完整的配音文本（用于 TTS 生成）
- **动画**：时间轴和动画效果描述
- **目的**：该分镜的教学目标

## 音频清单

最后生成音频文件清单表格：

| 幕号 | 文件名 | 读白文本 | 时长 | 说话人 | 情感 |
|------|--------|----------|------|--------|------|
| 1 | audio_001_scene.wav | 完整题目读白 | | Nofish | 平和 |
| 2 | audio_002_scene.wav | 条件分析读白 | | Nofish | 平和 |
| 3 | audio_003_scene.wav | 第 1 问求解读白 | | Nofish | 平和 |
| 4 | audio_004_scene.wav | 第 2 问求解读白 | | Nofish | 平和 |
| 5 | audio_005_scene.wav | 第 3 问求解读白 | | Nofish | 平和 |
| 6 | audio_006_scene.wav | 验证答案读白 | | Nofish | 热情 |
| 7 | audio_007_scene.wav | 方法总结读白 | | Nofish | 热情 |
| 8 | audio_008_scene.wav | 拓展思考读白 | | Nofish | 热情 |

## 注意事项

1. **读白文本**：使用口语化表达，避免书面语，便于 TTS 朗读
2. **数学公式**：读白中用中文读出公式（如"x 等于负 2a 分之 b"）
3. **时间控制**：每幕读白时长控制在 15-30 秒，总视频时长 5-6 分钟
4. **逻辑连贯**：各幕之间要有过渡，形成完整的解题流程
5. **重点突出**：关键步骤和易错点要强调
```

## 使用示例

### 输入
```python
problem_text = "已知抛物线 y = x² - (2m+4)x + m² + 4m。(1) 求顶点坐标；(2) A(-1,a), B(4,b) 在抛物线上，a>b，求 m 范围；(3) 在 -1≤x≤3 上，最大值与最小值之差为 4，求 m。"
problem_type = "抛物线综合题"
difficulty = 5
```

### 输出
完整的 8 幕分镜脚本（约 3000-4000 字符）

## 优化建议

1. **增加题型识别**：根据题目类型自动调整分镜结构
   - 几何题：增加图形绘制分镜
   - 应用题：增加建模分析分镜
   - 证明题：增加逻辑推理分镜

2. **增加难度适配**：根据难度调整讲解详细程度
   - 简单题：3-4 幕即可
   - 中等题：5-6 幕
   - 困难题：7-8 幕

3. **增加互动设计**：在关键步骤添加"思考时间"
   - "请同学们先思考一下..."
   - "你想到了吗？"

4. **增加错误分析**：针对常见错误添加警示
   - "注意：..."
   - "易错点：..."

5. **增加板书设计**：指定哪些内容用彩色标注
   - 关键公式用红色
   - 答案用绿色
   - 警示用黄色
