📐 抛物线问题详解

题目：已知抛物线 y = ax² + bx + 4 与x轴交于A、B两点（点A在点B的左侧），与y轴交于点C，点B(3, 0)，对称轴为直线 x = 1。

(1) 求该抛物线的解析式；
(2) 作直线BC，点P是抛物线上一动点，作直线PC，当∠PCB = ∠ABC时，求点P的坐标。

第(1)问：求解析式

步骤1：利用对称性求点A
对称轴 x = 1，B(3, 0)
A和B关于x=1对称：
(x_A + 3)/2 = 1 → x_A = -1
所以 A(-1, 0)
步骤2：代入点求系数
抛物线过A(-1, 0)：a - b = -4 ...①
抛物线过B(3, 0)：9a + 3b = -4 ...②

由①：b = a + 4，代入②：
9a + 3(a+4) = -4 → 12a = -16 → a = -4/3
b = -4/3 + 4 = 8/3
答案：y = -4/3 x² + 8/3 x + 4

图形可视化

第(2)问：求点P坐标

分析：点P在抛物线上，且∠PCB = ∠ABC

这意味着直线PC与BC的夹角等于AB与BC的夹角。

有两种情况：
1. PC与BC的夹角在BC上方
2. PC与BC的夹角在BC下方
答案：P₁(2, 4)，P₂(7/3, 20/9)
