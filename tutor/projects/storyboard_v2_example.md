**视觉文本**：
```
已知抛物线 \(y = x^2 - (2m+4)x + m^2 + 4m\)

(1) 求顶点坐标
(2) \(A(-1,a)\), \(B(4,b)\) 在抛物线上，\(a>b\)，求 \(m\) 范围
(3) 在 \(-1 \leq x \leq 3\) 上，最大值 - 最小值=4，求 \(m\)
```

**TTS 读白**：
<tts>已知抛物线 y 等于 x 平方减 2m 加 4 的和乘以 x，再加 m 平方加 4m。第一问，求顶点坐标；第二问，A 负 1 逗号 a，B4 逗号 b，a 大于 b，求 m 的取值范围；第三问，x 在负 1 到 3 之间，最大值减最小值等于 4，求 m 的值。</tts>

### 第 3 幕：第 1 问求解
**视觉文本**：
```
顶点公式：\(x = -\frac{b}{2a}\)

代入：\(x = -\frac{-(2m+4)}{2} = m+2\)

求 y：\(y = (m+2)^2 - (2m+4)(m+2) + m^2 + 4m = -4\)

∴ 顶点坐标：\((m+2, -4)\)
```

**TTS 读白**：
<tts>抛物线的顶点横坐标公式是 x 等于负 2a 分之 b。这里 a 等于 1，b 等于负 2m 减 4。代入公式，x 等于负 2 分之负 2m 减 4，化简后得到 x 等于 m 加 2。将 x 等于 m 加 2 代入原方程，化简后得到 y 等于负 4。所以顶点坐标是 m 加 2 逗号负 4。</tts>

**画面**：展示顶点公式和代入过程
**字幕**："第 (1) 问：求顶点坐标"
**动画**：
- 0.0s: 显示顶点公式
- 1.5s: 代入 a,b 值
- 3.0s: 化简得 x=m+2
- 4.5s: 代入求 y
- 6.0s: 显示最终结果
**目的**：展示顶点坐标的完整推导过程

### 第 4 幕：第 2 问求解
**视觉文本**：
```
代入 A(-1, a)：
\(a = (-1)^2 - (2m+4)(-1) + m^2 + 4m = m^2 + 6m + 5\)

代入 B(4, b)：
\(b = 4^2 - (2m+4)(4) + m^2 + 4m = m^2 - 4m\)

由 a > b：
\(m^2 + 6m + 5 > m^2 - 4m\)
\(10m > -5\)
\(m > -\frac{1}{2}\)
```

**TTS 读白**：
<tts>将 A 点横坐标负 1 代入抛物线方程，得到 a 等于 m 平方加 6m 加 5。将 B 点横坐标 4 代入，得到 b 等于 m 平方减 4m。因为 a 大于 b，所以 m 平方加 6m 加 5 大于 m 平方减 4m。化简得 10m 大于负 5，所以 m 大于负 2 分之 1。</tts>

**画面**：展示代入 A、B 点和解不等式
**字幕**："第 (2) 问：求 m 范围"
**动画**：
- 0.0s: 代入 A 点
- 2.0s: 化简得 a 的表达式
- 4.0s: 代入 B 点
- 6.0s: 化简得 b 的表达式
- 8.0s: 列出不等式
- 10.0s: 解出 m 的范围
**目的**：展示代入法和不等式求解过程

### 第 5 幕：第 3 问求解
**视觉文本**：
```
对称轴：\(x = m+2\)
区间：\(-1 \leq x \leq 3\)
开口向上（\(a=1>0\)）

分类讨论：
Case 1: \(m < -3\) → min at x=-1, max at x=3
Case 2: \(m > 1\) → min at x=3, max at x=-1
Case 3: \(-3 \leq m \leq 1\) → min at vertex, max at endpoint
```

**TTS 读白**：
<tts>对称轴是 x 等于 m 加 2，给定区间是负 1 到 3。因为抛物线开口向上，需要分类讨论。当 m 小于负 3 时，最小值在 x 等于负 1，最大值在 x 等于 3。当 m 大于 1 时，最小值在 x 等于 3，最大值在 x 等于负 1。当 m 在负 3 到 1 之间时，最小值在顶点，最大值在端点。根据最大值减最小值等于 4，可以求出 m 的值。</tts>

**画面**：展示对称轴位置和分类讨论
**字幕**："第 (3) 问：最大值 - 最小值 = 4"
**动画**：
- 0.0s: 显示对称轴公式

- 7.0s: 情况 2: m>1
- 9.0s: 情况 3: -3≤m≤1
- 11.0s: 说明求解方法
**目的**：展示分类讨论思想和解题思路
