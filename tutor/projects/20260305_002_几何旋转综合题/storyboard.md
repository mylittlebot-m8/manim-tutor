### 第 3 幕：第 1 问 - 完成证明
**视觉文本**：
```
证明：

在△ACD 和△BED 中：
CD = DE（已知）
∠ADC = ∠BDE（对顶角相等）
AD = BD（待证）

∵ ∠ACB = 90°，∠CDE = 90°
∴ ∠ACD + ∠BCD = 90°
   ∠BDE + ∠CDE = 90°

又∵ ∠BCD = ∠CDE（旋转）
∴ ∠ACD = ∠BDE

∴ △ACD ≌ △BED（SAS）

∴ ∠CAD = ∠EBD

∵ ∠CAD + ∠CBA = 90°（Rt△ABC）
∴ ∠EBD + ∠CBA = 90°

即 ∠ABE = 90°  得证
```

**TTS 读白**：
<tts>证明：在三角形 ACD 和三角形 BED 中，CD 等于 DE，这是已知条件。角 ADC 等于角 BDE，这是对顶角相等。因为角 ACB 等于 90 度，角 CDE 等于 90 度，所以角 ACD 加角 BCD 等于 90 度，角 BDE 加角 CDE 等于 90 度。又因为角 BCD 等于角 CDE，这是旋转得到的，所以角 ACD 等于角 BDE。所以三角形 ACD 全等于三角形 BED，依据是边角边定理。所以角 CAD 等于角 EBD。因为角 CAD 加角 CBA 等于 90 度，这是直角三角形 ABC，所以角 EBD 加角 CBA 等于 90 度，即角 ABE 等于 90 度，得证。</tts>

### 第 7 幕：第 3 问① - 完成证明
**视觉文本**：
```
证明：

在 Rt△CDE 中：
CE² = CD² + DE²（勾股定理）
    = CD² + CD²（∵ CD = DE）
    = 2CD²

证明 CD² = BD·AD：

在△ACD 和△CBD 中：
∠A = ∠BCD（同角的余角相等）
∠ADC = ∠CDB（公共角）

∴ △ACD ∽ △CBD（AA）

∴ AD/CD = CD/BD（相似三角形对应边成比例）

即 CD² = BD·AD

∴ CE² = 2CD² = 2BD·AD  得证
```

**TTS 读白**：
<tts>证明：在直角三角形 CDE 中，CE 的平方等于 CD 的平方加 DE 的平方，这是勾股定理，等于 CD 的平方加 CD 的平方，因为 CD 等于 DE，等于 2 乘以 CD 的平方。证明 CD 的平方等于 BD 乘以 AD：在三角形 ACD 和三角形 CBD 中，角 A 等于角 BCD，这是同角的余角相等，角 ADC 等于角 CDB，这是公共角，所以三角形 ACD 相似于三角形 CBD，依据是角角定理。所以 AD 比 CD 等于 CD 比 BD，这是相似三角形对应边成比例，即 CD 的平方等于 BD 乘以 AD。所以 CE 的平方等于 2 乘以 CD 的平方等于 2 乘以 BD 乘以 AD，得证。</tts>

### 第 8 幕：第 3 问② - 求 CE 最小值
**视觉文本**：
```
第 (3) 问②：求 CE 的最小值

由①知：CE² = 2BD·AD

设 AD = x，则 BD = 5 - x
（∵ AB = 5）

CE² = 2(5-x)·x
    = 2(5x - x²)
    = -2x² + 10x

这是关于 x 的二次函数
开口向下，有最大值

当 x = -b/(2a) = -10/(-4) = 2.5 时
CE² 有最大值 = -2(2.5)² + 10(2.5) = 12.5

∴ CE 的最小值 = √12.5 = 5√2/2

此时 D 为 AB 中点
```

**TTS 读白**：
<tts>第三问第二小问，求 CE 的最小值。由第一小问知，CE 的平方等于 2 乘以 BD 乘以 AD。设 AD 等于 x，则 BD 等于 5 减 x，因为 AB 等于 5。CE 的平方等于 2 乘以 5 减 x 乘以 x，等于 2 乘以 5x 减 x 的平方，等于负 2x 的平方加 10x。这是关于 x 的二次函数，开口向下，有最大值。当 x 等于负 b 除以 2a 等于负 10 除以负 4 等于 2.5 时，CE 的平方有最大值等于负 2 乘以 2.5 的平方加 10 乘以 2.5 等于 12.5。所以 CE 的最小值等于根号 12.5 等于 2 分之 5 倍根号 2。此时 D 为 AB 中点。</tts>
