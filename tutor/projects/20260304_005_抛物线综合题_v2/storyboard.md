# 分镜脚本 - 抛物线综合题（V2 格式）

## 分镜设计

### 第 1 幕：题目展示
**视觉文本**：
```
抛物线 y = -x² + 2x + 3

(1) 求 A、B、C、D 坐标
(2) 求△PAC 面积最大时的 P 点
(3) 对称轴上是否存在点 Q，使△QBC 为等腰三角形？
```

**TTS 读白**：
<tts>在平面直角坐标系中，抛物线 y 等于负 x 平方加 2x 加 3，与 x 轴交于 A、B 两点，点 A 在左侧，与 y 轴交于点 C，顶点为 D。第一问，求点 A、B、C、D 的坐标；第二问，点 P 是第一象限内抛物线上一动点，当三角形 PAC 的面积最大时，求点 P 的坐标及最大面积；第三问，在抛物线的对称轴上是否存在点 Q，使得以 Q、B、C 为顶点的三角形是等腰三角形？</tts>

**画面**：深色背景，题目文字逐行显示
**字幕**："抛物线综合题"
**动画**：
- 0.0s: 标题淡入

- 2.0s: 三个问题逐行显示
**目的**：引入题目

### 第 2 幕：第 1 问 - 求 A、B 坐标
**视觉文本**：
```
令 y = 0：
-x² + 2x + 3 = 0
x² - 2x - 3 = 0
(x-3)(x+1) = 0

∴ x₁ = -1, x₂ = 3
A(-1, 0), B(3, 0)
```

**TTS 读白**：
<tts>第一问，求 A、B、C、D 的坐标。首先求 A、B 坐标，令 y 等于 0，得到方程负 x 平方加 2x 加 3 等于 0，整理得 x 平方减 2x 减 3 等于 0，因式分解得 x 减 3 乘以 x 加 1 等于 0，解得 x 一等于负 1，x 二等于 3。因为点 A 在左侧，所以 A 点坐标是负 1 逗号 0，B 点坐标是 3 逗号 0。</tts>

**画面**：方程求解过程逐步显示
**字幕**："第 (1) 问：求 A、B 坐标"
**动画**：
- 0.0s: 令 y=0
- 1.5s: 方程显示
- 3.0s: 因式分解
- 4.5s: 显示 A、B 坐标
**目的**：展示求根过程

### 第 3 幕：第 1 问 - 求 C、D 坐标
**视觉文本**：
```
求 C 点（与 y 轴交点）：
令 x = 0：y = 3
∴ C(0, 3)

求顶点 D：
x = -b/(2a) = -2/(-2) = 1
y = -1 + 2 + 3 = 4
∴ D(1, 4)
```

**TTS 读白**：
<tts>求 C 点坐标，C 是与 y 轴的交点，令 x 等于 0，代入得 y 等于 3，所以 C 点坐标是 0 逗号 3。求顶点 D 的坐标，顶点横坐标公式是 x 等于负 2a 分之 b，代入得 x 等于 1，将 x 等于 1 代入原方程，得 y 等于负 1 加 2 加 3 等于 4，所以顶点 D 的坐标是 1 逗号 4。</tts>

**画面**：C 点和顶点 D 的计算过程
**字幕**："求 C、D 坐标"
**动画**：
- 0.0s: 求 C 点

**目的**：完成第 1 问

### 第 4 幕：第 2 问 - 面积公式
**视觉文本**：
```
设 P(x, -x²+2x+3)，0 < x < 3

S△PAC = ½ |xA(yC-yP) + xC(yP-yA) + xP(yA-yC)|
       = ½ |-1(3-yP) + 0 + x(0-3)|
       = ½ |-3 + yP - 3x|
```

**TTS 读白**：
<tts>第二问，求三角形 PAC 面积最大时的 P 点坐标。设 P 点坐标为 x 逗号负 x 平方加 2x 加 3，其中 x 大于 0 小于 3。使用三角形面积坐标公式，S 三角形 PAC 等于二分之一乘以 xA 乘以 yC 减 yP，加 xC 乘以 yP 减 yA，加 xP 乘以 yA 减 yC 的绝对值。代入 A 负 1 逗号 0，C0 逗号 3，化简后得到面积表达式。</tts>

**画面**：面积公式推导
**字幕**："第 (2) 问：求面积最大值"
**动画**：
- 0.0s: 设 P 点坐标
- 2.0s: 面积公式
- 4.0s: 代入化简
**目的**：建立面积函数

### 第 5 幕：第 2 问 - 求最大值
**视觉文本**：
```
S = ½ |-3 + (-x²+2x+3) - 3x|
  = ½ |-x² - x|
  = ½ (x² + x)  (∵ x > 0)

求导：S' = ½ (2x + 1)
令 S' = 0：x = -½ (舍去)

∴ 在 (0,3) 内，S 单调递增
当 x = 3 时，Smax = ½ (9+3) = 6
此时 P(3, 0)
```

**TTS 读白**：
<tts>将 P 点纵坐标代入面积公式，化简得 S 等于二分之一乘以负 x 平方减 x 的绝对值。因为 x 大于 0，所以 S 等于二分之一乘以 x 平方加 x。对 S 求导，得 S 撇等于二分之一乘以 2x 加 1。令导数等于 0，解得 x 等于负 2 分之 1，但在定义域外，舍去。所以在 0 到 3 的区间内，S 单调递增。当 x 等于 3 时，面积最大，最大值为 6，此时 P 点坐标是 3 逗号 0。</tts>

**画面**：求导和最大值计算
**字幕**："求面积最大值"
**动画**：
- 0.0s: 化简面积公式
- 2.0s: 求导
- 4.0s: 求最大值
- 6.0s: 显示 P 点坐标
**目的**：完成第 2 问

### 第 6 幕：第 3 问 - 等腰三角形分析
**视觉文本**：
```
对称轴：x = 1
设 Q(1, y)

B(3, 0), C(0, 3)

QB² = (3-1)² + (0-y)² = 4 + y²
QC² = (0-1)² + (3-y)² = 1 + (3-y)²
BC² = (3-0)² + (0-3)² = 18
```

**TTS 读白**：
<tts>第三问，在对称轴上是否存在点 Q，使三角形 QBC 为等腰三角形。对称轴是 x 等于 1，设 Q 点坐标为 1 逗号 y。B 点坐标是 3 逗号 0，C 点坐标是 0 逗号 3。计算 QB 的平方等于 4 加 y 平方，QC 的平方等于 1 加 3 减 y 的平方，BC 的平方等于 18。分三种情况讨论：QB 等于 QC，QB 等于 BC，或 QC 等于 BC。</tts>

**画面**：距离公式计算
**字幕**："第 (3) 问：等腰三角形存在性"
**动画**：
- 0.0s: 设 Q 点坐标
- 1.5s: 计算 QB²
- 3.0s: 计算 QC²
- 4.5s: 计算 BC²
**目的**：建立方程基础

### 第 7 幕：第 3 问 - 三种情况
**视觉文本**：
```
情况 1：QB = QC
4 + y² = 1 + (3-y)²
解得：y = 1  ∴ Q₁(1, 1)

情况 2：QB = BC
4 + y² = 18
解得：y = ±√14  ∴ Q₂(1, √14), Q₃(1, -√14)

情况 3：QC = BC
1 + (3-y)² = 18
解得：y = 3±√17  ∴ Q₄(1, 3+√17), Q₅(1, 3-√17)
```

**TTS 读白**：
<tts>情况一，QB 等于 QC，列方程 4 加 y 平方等于 1 加 3 减 y 的平方，解得 y 等于 1，所以 Q 一坐标是 1 逗号 1。情况二，QB 等于 BC，列方程 4 加 y 平方等于 18，解得 y 等于正负根号 14，所以 Q 二坐标是 1 逗号根号 14，Q 三坐标是 1 逗号负根号 14。情况三，QC 等于 BC，列方程 1 加 3 减 y 的平方等于 18，解得 y 等于 3 加减根号 17，所以 Q 四坐标是 1 逗号 3 加根号 17，Q 五坐标是 1 逗号 3 减根号 17。</tts>

**画面**：三种情况的方程求解
**字幕**："三种情况讨论"
**动画**：
- 0.0s: 情况 1
- 2.0s: 情况 2
- 4.0s: 情况 3
- 6.0s: 汇总所有 Q 点
**目的**：完成第 3 问

**视觉文本**：
```
答案汇总：
(1) A(-1,0), B(3,0), C(0,3), D(1,4)
(2) P(3,0), Smax = 6
(3) 存在 5 个点 Q：
    (1,1), (1,±√14), (1,3±√17)
```
